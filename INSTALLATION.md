# Installation Guide

## Quick Start

The easiest way to install SickoMenu is using pip:

```bash
pip install sickomenu
```

## From Source

```bash
git clone https://github.com/MRuhan17/SickoMenu.git
cd SickoMenu
pip install -e .
```

## Requirements

- Python 3.8+
- pip

## Optional Dependencies

For development:

```bash
pip install -r requirements-dev.txt
```

## Verification

Verify the installation:

```bash
python -c "import sickomenu; print(sickomenu.__version__)"
```
