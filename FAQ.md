# Frequently Asked Questions

## General Questions

### What is SickoMenu?

SickoMenu is a Python-based project designed to provide menu functionality with various features and customizations.

### How do I install SickoMenu?

See INSTALLATION.md for detailed installation instructions.

### What Python versions are supported?

SickoMenu supports Python 3.8 and newer.

## Development Questions

### How can I contribute?

See CONTRIBUTING.md for guidelines on how to contribute to the project.

### How do I report a bug?

1. Check if the bug has already been reported
2. Use the bug report template in GitHub Issues
3. Provide detailed reproduction steps

### How do I request a feature?

Use the feature request template in GitHub Issues with:
- Clear description of the feature
- Use cases
- Examples

## Usage Questions

### Where can I find documentation?

Documentation is available in the repository:
- DEVELOPMENT.md - Development setup
- BUILD.md - Build instructions
- TESTING.md - Testing guide

### How do I get help?

1. Check TROUBLESHOOTING.md
2. Search GitHub Issues
3. Create a new issue with details

## Technical Questions

### How are dependencies managed?

Dependencies are managed through requirements.txt and can be automatically updated with Dependabot.

### How is the code formatted?

We use Black for code formatting and Flake8 for linting. See DEVELOPMENT.md for details.
